## 🚀 Quick start

1.  Build the Gatsby site (for development).

    Use the Gatsby CLI to create the site

    ```shell
    # Creates a basic localhost server, and hosts the Gatsby site.
    gatsby develop
    ```

    You will now have a new server running at http://localhost:8000


## 💫 Full Deployment

If you want to build this for production, simply use the following command:

    ```shell
    # Builds your full Gatsby site.
    gatsby build
    ```

This will consume the data from the GraphQL interface, and render a static site in the `/static` directory.

You can then point your server to that directory and **viola!**
